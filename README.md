# homebrew-tap

Homebrew formulae for personal projects.

## Install

```bash
brew tap mikevanl/tap
```

## Formulae

| Formula | Description |
|---|---|
| `wally` | Native macOS video wallpaper manager with CLI |

### Wally

```bash
brew install mikevanl/tap/wally
```

See [Wally](https://github.com/mikevanl/Wally) for full documentation.
