# homebrew-tap

Homebrew formulae for personal projects.

## Install

```bash
brew tap mikevanl/tap
```

## Formulae

| Formula | Description |
|---|---|
| `wally` | Native macOS video wallpaper manager with CLI |
| `audiomix` | Per-app volume, mute, and output device routing for macOS |

### Wally

```bash
brew install mikevanl/tap/wally
```

See [Wally](https://github.com/mikevanl/Wally) for full documentation.

### AudioMix

```bash
brew install mikevanl/tap/audiomix
```

See [AudioMix](https://github.com/mikevanl/audiomix) for full documentation.
